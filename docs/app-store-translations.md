## Translation Notes

### Localization Considerations

1. **Date Formats**
   - US: MM/DD/YYYY
   - EU: DD/MM/YYYY
   - Japan: YYYY/MM/DD
   - Ensure app handles regional date preferences

2. **Currency for Tickets**
   - Display prices in local currency
   - Use appropriate currency symbols

3. **Cultural Adaptations**
   - Germany: Emphasize privacy and data security
   - Japan: Highlight precision and efficiency
   - Spain/Latin America: Emphasize social sharing
   - US: Focus on time-saving and productivity

4. **Screenshot Text**
   - All screenshots should be localized
   - Use real events from target regions
   - Show local venues and familiar event types

5. **Support Resources**
   - Provide support in each language
   - Create localized FAQs
   - Hire native speakers for app review responses

### ASO Best Practices

1. **App Name**: Include "EventSnap" + primary keyword if space allows
2. **Subtitle/Short Description**: Must contain 2-3 high-value keywords
3. **Long Description**: Front-load keywords in first 250 characters
4. **Keywords**: Avoid duplicating app name words
5. **Updates**: Update keywords based on search performance data

### Testing Requirements

- [ ] Native speakers review all translations
- [ ] Test all text renders properly in UI (no truncation)
- [ ] Verify keywords perform in local markets
- [ ] A/B test different app names per region
- [ ] Monitor conversion rates by language
